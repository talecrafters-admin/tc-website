## üé® Design Guidelines

### Color Usage

1. **Transformative Teal (Primary):**

   - Logo backgrounds
   - Primary buttons (non-CTA)
   - Hover states
   - Success messages

2. **Persimmon (CTA):**

   - All "Get Free Audit" buttons
   - Main call-to-action elements
   - Urgent notifications

3. **Wasabi (Secondary):**

   - Badge backgrounds
   - Secondary highlights
   - Playful accents

4. **High-Voltage Yellow:**
   - Warning states
   - Electric accents
   - Hover effects

### Typography Rules

- **ALL CAPS** for hero headlines
- `font-heading` class for Black Ops One
- `tracking-tight` for condensed impact
- `leading-[0.9]` for dramatic line height
- `text-stroke-sm` for outlined text

### Spacing

- Larger gaps between elements
- More breathing room
- Generous padding on cards
