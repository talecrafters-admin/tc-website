# Client Logos Folder

This folder is for storing client logos to display on the website.

## Instructions

1. **File Format**: PNG with transparent background (preferred)
   - Acceptable formats: PNG, SVG, WEBP
   - Avoid: JPG (unless no transparency needed)

2. **File Naming**: Use lowercase with hyphens
   - Good: `acme-corporation.png`, `tech-startup.png`
   - Bad: `Acme Corporation.PNG`, `client_1.jpg`

3. **Image Specifications**:
   - Recommended size: 200x100px (width x height)
   - Max file size: 100KB per logo
   - High resolution for retina displays

4. **Number of Logos**: Add 25+ client logos
   - The website is designed to showcase all your clients
   - Logos will be displayed in a grid/carousel format

5. **Logo Quality**:
   - Use official brand logos when possible
   - Ensure logos are clear and legible
   - Test on both light and dark backgrounds

## Display Format

Logos will be displayed:
- In grayscale by default
- In color on hover
- With subtle animations
- In a responsive grid layout
